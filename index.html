Pilas (Stacks): Uso de LIFO

Estructura de datos lineal que funciona con el principio LIFO (Last In, First Out), es decir, el último elemento que se introduce es el primero que se elimina.

Información sobre Pilas

El Iterador: TOPE

Es el único punto de acceso de la pila. Indica la posición del último elemento insertado. Todas las operaciones (PUSH/POP) se realizan a través del Tope.

Error: DESBORDAMIENTO (Overflow)

Este error Ocurre cuando se intenta hacer un PUSH en una pila que ya ha alcanzado su capacidad máxima. En esta simulación, la capacidad máxima es de 5 elementos

Error: SUBDESBORDAMIENTO (Underflow)

Ocurre cuando se intenta hacer un POP o PEEK (identificar el tope) en una pila que está vacía. No hay elementos para extraer o consultar.

Las pilas se usan comúnmente en programación para manejar llamadas a funciones, evaluar expresiones y controlar la navegación en navegadores web, entre otras aplicaciones.
